Click-Describe: Multimodal Grounding and Tracking for Aerial Objects

A research project exploring multimodal object grounding and tracking.

Read the Paper View Code

Abstract

This project investigates a novel approach to multimodal grounding and tracking for aerial objects. Our method enables users to click on an object in an aerial image and generate a textual description while tracking it across frames.
